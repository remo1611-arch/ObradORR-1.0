# Fase 6.72.1 · Shell docente limpio por dominios

Navegación principal nueva:

- Taller
- Histórico
- Archivo técnico
- Sistema

## Criterio

La UI visible se organiza por dominio docente, no por módulos históricos de implementación.

- Taller: práctica activa, pedido e impresión.
- Histórico: prácticas archivadas o sesiones cerradas.
- Archivo técnico: elaboraciones, ingredientes, subelaboraciones y revisión.
- Sistema: copias, importación/exportación, recuperación y SQL técnico.

## Alcance

No se modifican cálculo, SQLite, contratos, recursividad, print-service, persistencia ni base de fichas. Es una sustitución de shell y navegación con mapeo de áreas, manteniendo el motor estable.
