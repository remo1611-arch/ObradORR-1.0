# Aviso de autoría

**ObradORR 1.0.0**

Autor y titular: **Remo José Pereira González**.

Aplicación docente para la organización, consulta, edición, escalado e impresión de fichas técnicas de cocina, pastelería y panadería.

El catálogo incluido constituye una base documental de trabajo. Las fichas no se consideran validadas por el mero hecho de formar parte del programa.
