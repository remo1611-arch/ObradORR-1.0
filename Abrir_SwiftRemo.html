🍽️🍞 Abrir SwiftRemo

Este archivo es un lanzador local. Guarda la URL pública de GitHub Pages y abre siempre la versión publicada de SwiftRemo desde cualquier dispositivo.

La URL puede ser la raíz del sitio, por ejemplo https://usuario.github.io/swiftremo/, o directamente https://usuario.github.io/swiftremo/app/sqlite.html?v=6731.

Los datos del usuario se guardan en su navegador. Para enviar cambios o fichas, debe descargar una copia .sqlite.
